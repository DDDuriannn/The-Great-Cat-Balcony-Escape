# 🐱 猫咪阳台大逃亡 2.0

一只小猫在居民楼的阳台、空调外机、晾衣杆之间奔跑跳跃，**躲避障碍、收集鱼干、与狗狗斗智斗勇**的轻策略网页跑酷小游戏。2.0 在前作基础上加入了 6 个新系统：三层路线、狗追逐容错、鱼干能量冲刺、随机事件、短目标任务、猫咪状态。

> 项目代号：`cat-balcony-escape`
> 双击 `index.html` 即可开始游戏。

---

## ✨ 特性

- ✅ **零依赖**：纯 HTML + CSS + 原生 JavaScript + Canvas 2D，无后端、无数据库、无构建工具、无外部素材
- 🎨 **Canvas 程序化绘制**：小猫、障碍、背景、城市居民楼全部用代码绘制
- 🔊 **Web Audio API 合成音效**：跳跃 / 鱼干 / 撞击 / 冲刺 / 任务完成 / 事件预警 全部程序合成
- 📱 **响应式**：自适应桌面与移动端，支持键盘 + 触屏
- 💾 **本地存档**：最高分、累计鱼干、静音状态保存在 `localStorage`
- 🚀 **适合 GitHub Pages**：纯静态文件，无需任何后端
- 🎮 **6 大新系统**：
  - 🛣️ **三层路线**（上层 / 中层 / 下层）—— 不同鱼干密度、风险与平台风格
  - 🐕 **狗追逐容错** —— 危险度条（0-100%）取代一击即死，撞狗只增危险度
  - ⚡ **鱼干能量 / 冲刺** —— 收集鱼干充能，满了按跳跃触发 3 秒冲刺（速度 ×1.8、无敌、吸附）
  - 🎲 **6 种随机事件**（掉落花盆 / 伸出扫把 / 晃动晾衣杆 / 空调喷风 / 狗叫区 / 关窗）
  - 🎯 **短目标任务** —— 每局随机抽 3 个任务，完成获得奖励
  - 😺 **6 种猫咪状态**（正常 / 炸毛 / 馋猫 / 灵巧 / 护盾 / 冲刺）

## 🎮 如何运行

### 方式 A：双击运行（最简单）

直接双击 `index.html`，浏览器打开即可开始。

### 方式 B：本地 HTTP 服务器（推荐）

```bash
# Python 3
python3 -m http.server 8000

# 或者 Node.js
npx serve .
```

然后访问 `http://localhost:8000`。

## 🕹 玩法详解

### 三层路线

小猫可在三层楼之间穿梭。跳跃高度（单跳 ≈ 130px，二段跳 ≈ 240px）正好支持跨层跳跃。

| 路线 | Y | 平台宽 | 鱼干 | 主要风险 |
|------|---|--------|------|----------|
| **上层** | 290 | 60-160 | 50% 概率金鱼干 | 关窗、掉落花盆、晃动晾衣杆 |
| **中层** | 420 | 120-280 | 普通鱼干 | 扫把、水桶、狗叫区 |
| **下层** | 500 | 100-220 | 普通鱼干 + 罐头 | 空调喷风、真实缺口 |

**真实缺口**：在 lower lane 距离 > 400m 后随机出现，玩家必须跳过去，否则直接 game over。

### 狗追逐容错

屏幕左侧的垂直 **危险度条** 取代了原来的"撞到就死"。

- 默认 `0%`
- 轻碰撞（扫把 / 水桶 / 花盆 / 关窗）：`+12%`
- 重碰撞（不挡狗的障碍）：`+25%`
- 进入狗叫区 / 狗叫事件：`+20%` + 进入 **炸毛** 状态
- 掉落花盆：`+15%`
- 掉入缺口 / 摔出屏幕：立即 game over（保留硬失败）
- 收集普通鱼干：`-2%`
- 收集金鱼干：`-5%`
- 完美落地：`-3%`
- 冲刺期间：狗被甩开 `-30%`
- `100%` → game over

**UI 反馈**：
- 50%+ → 屏幕边缘红色脉动（vignette）
- 80%+ → 屏幕轻微持续震动

### 鱼干能量 / 冲刺

| 收集物 | 加分 | 加能量 | 加 / 减危险 | 备注 |
|--------|------|--------|-------------|------|
| 🐟 普通鱼干 | +5 | +5 | -2% | 上层 50% 概率出 |
| 🌟 金鱼干 | +25 | +20 | -5% | 集中在上层 |
| 🥫 罐头 | +10 | —— | -30% | 触发 5 秒 **护盾** |

能量条满（100）时按跳跃触发 **冲刺** 3 秒：
- 速度 × 1.8
- **无敌**（撞击不减危险度 / 不死）
- 鱼干吸附（200px 内自动吸过来）
- 狗被甩开 `-30%` 危险度
- 冲刺后能量归零
- 屏幕速度线粒子

### Combo

2 秒内每收一个鱼干：combo +1（普通 +1，金鱼干 +2）。倍率随 combo 跳变：
`1x`（默认）→ `1.2x`（5）→ `1.5x`（10）→ `2x`（20）→ `3x`（30）
连击 ≥ 8 触发 **馋猫** 状态（80px 鱼干磁吸）。

### 短目标任务

每局开始随机抽 3 个任务（10 个模板），进度条 + ✓ 完成动画。

任务类型：
1. `collect_fish` 收集 12 条鱼干
2. `collect_gold` 收集 3 个金鱼干
3. `combo_8` 达成 8 连击
4. `distance_300` / `distance_500` 跑过 300/500 米
5. `upper_lane_100` 上层奔跑 100 米
6. `lower_lane_50` 使用下层 50 米
7. `perfect_3` 完美落地 3 次
8. `dash_once` 触发 1 次冲刺
9. `dodge_flowerpot` 躲过 5 个花盆

奖励类型：
- `score`：+50~200 立即加分
- `energy`：+30~50 能量
- `danger`：-20~30% 危险度
- `multiplier`：该局鱼干分 +50%

### 6 种随机事件

| 事件 | 解锁距离 | 预警时长 | 触发效果 |
|------|----------|----------|----------|
| 🌸 掉落花盆 | 150m | 1.0s（阴影） | 从屏幕上方落下 |
| 🧹 伸出扫把 | 200m | 0.5s（伸长） | 横向扫过 |
| 👕 晃动晾衣杆 | 450m | 1.5s（抖动） | 踩上去额外受伤 |
| ❄️ 空调喷风 | 500m | 1.0s（气流） | 强制把玩家向上推 100px |
| 🐕 狗叫区 | 800m | 持续显示 | 进入受伤 + 炸毛 + 狗叫 |
| 🪟 关窗 | 900m | 1.5s（缓慢关闭） | 关后撞墙 |

每个事件都有视觉预警，至少 0.5s 反应时间。

### 6 种猫咪状态

| 状态 | 触发 | 持续 | 效果 |
|------|------|------|------|
| 🐾 正常 | 默认 | —— | —— |
| 😾 炸毛 | 进入狗叫区 / 危险度 ≥ 70% | 4s | 速度 ×1.1 |
| 😻 馋猫 | 连续 8 combo | 5s | 80px 鱼干磁吸 |
| 🐾 灵巧 | 连续 3 次完美落地 | 5s | 跳跃高度 ×1.3 |
| 🛡️ 护盾 | 吃罐头 / 任务奖励 | 5s | 抵消一次危险度增加 |
| ⚡ 冲刺 | 能量满触发 | 3s | 速度 ×1.8、无敌、吸附 |

冲刺 / 护盾状态有最高优先级，不被普通状态覆盖。

## 🛠 状态机

```
loading → ready → playing ⇄ paused → gameOver → ready
```

- `loading` 资源准备（极短）
- `ready` 开始界面，显示历史最高 + 累计鱼干
- `playing` 正常运行（HUD + 任务 + 能量 + 危险度条）
- `paused` 暂停（失焦自动暂停）
- `gameOver` 结束，显示本局分数、最高分、新纪录提示、任务完成数

## 🐞 常见问题

**Q：双击 index.html 没有声音？**
A：浏览器对 `file://` 下的 `AudioContext` 创建有限制。点击页面任意位置后，浏览器才会激活音频。或者用 `python3 -m http.server` 起本地服务后再访问。

**Q：移动端点击没反应？**
A：检查浏览器是否启用了 JavaScript。部分浏览器（如微信内置）需要先点右上角菜单"在浏览器中打开"。

**Q：游戏太难 / 太简单？**
A：所有参数都在 `game.js` 顶部的 `Config` 对象里。例如：
- `SPEED_MAX`：最高速度（默认 720）
- `DAMAGE_LIGHT` / `DAMAGE_HEAVY`：碰撞伤害
- `EVENT_UNLOCK_DIST`：各事件解锁距离
- `PLATFORM_WIDTH` / `PLATFORM_GAP`：平台参数

**Q：怎么关闭某个系统？**
A：删除 `game.js` 中相应模块的 update 调用即可。例如不想要冲刺：删除 `Game._handleInput` 中的 `this._triggerDash()`。

## 🧩 后续可扩展方向

按实现成本从低到高排列：

1. **更多皮肤 / 角色**：黑猫、橘猫、奶牛猫 —— 给 `Player.draw` 加 `skin` 参数
2. **道具系统**：磁铁道具（永久吸附）、火箭道具（短时加速）
3. **关卡主题**：黄昏、雨夜、雾天 —— 给 `Background` 加主题枚举
4. **每日挑战种子**：用日期作为随机种子生成固定关卡
5. **在线排行榜**：用 Google Forms / Sheets 模拟后端
6. **PWA 离线安装**：加 `manifest.json` 和 Service Worker
7. **手柄支持**：用 Gamepad API 监听手柄输入
8. **i18n 多语言**：把所有文字抽到 `lang.js`，支持中 / 英切换
9. **回放系统**：记录每局输入，结束后可回放
10. **Boss 战**：每 1000m 一个 Boss（如巨型狗 / 鹰 / 乌鸦）
